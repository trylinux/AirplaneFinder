Usage Examples

Reading data (any permission level)

curl -H "Authorization: Bearer amt_YOUR_KEY_HERE" \
     http://localhost:5000/api/v1/aircraft/search?q=C-130

Creating a record (readwrite or admin)

curl -X POST -H "Authorization: Bearer amt_YOUR_KEY_HERE" \
     -H "Content-Type: application/json" \
     -d '{"manufacturer":"Boeing","model":"B-52","variant":"H"}' \
     http://localhost:5000/api/v1/aircraft

Deleting a record (admin only)

curl -X DELETE -H "Authorization: Bearer amt_YOUR_KEY_HERE" \
     http://localhost:5000/api/v1/aircraft/42
